# External Validation Kit

This kit scaffolds a standalone benchmark repository for objective external validation.
It supports true installed-skill execution for skill-based pipelines using `npx skills add`.

## Generate a New External Repo

```bash
bash benchmarks/comparison/external-validation-kit/scaffold-external-repo.sh \
  --target /absolute/path/skill-benchmark-lab
```

Then follow `/absolute/path/skill-benchmark-lab/RUNBOOK.md`.
